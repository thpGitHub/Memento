# Shortcuts Visual Studio Code

````shell
ctrl + L # selectionner toute une ligne
ctrl + X # supprime toute la ligne
ctrl + shift + Enter # insert une ligne au dessus
ctrl + shift # insert une ligne après

alt + Flèche Haut ou Flèche Bas # Déplace la ligne vers le haut ou le bas
alt + shift + fleche Haut ou bas # copier le ligne
````
